read
====

> reads from standard input into a variable _name_

- **read** [_name_]

---

[POSIX Specification](https://pubs.opengroup.org/onlinepubs/9799919799/utilities/read.html)

Basic Usage
-----------

Example:

    read my_var <<@
    foo
    @
    echo $my_var

Output:

    foo
