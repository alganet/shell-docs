unset
=====

> removes a variable declaration by _name_ if it exists

- **unset** _name_ [_name_...]

Basic Usage
-----------

Example:

    my_var="foo"
    echo Value: $my_var
    unset my_var
    echo Value: $my_var

Output:

    Value: foo
    Value:
